- feat(vite): Port babel-plugin-cedar-graphql-options-extract to Vite (#2070) by @lisa-assistant

Adds `vite-plugin-cedar-graphql-options-extract`, enabling downstream Vite plugins like `gqlorm-inject` to mutate shared state before the GraphQL handler is called.

The plugin extracts the options argument from `createGraphQLHandler` calls into an exported variable (`__cedar_graphqlOptions`), preserving the original code structure while enabling plugin composition.
