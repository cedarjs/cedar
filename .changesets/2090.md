- chore(vite): Use Vite import-dir plugin in buildApp (#2090) by @lisa-assistant

Integrates the existing `cedarImportDirPlugin` into the buildApp pipeline for API builds. This plugin handles glob-based imports (e.g., `import x from 'src/dir/**/*.ts'`) and is functionally equivalent to the previous Babel plugin-redwood-import-dir approach.

Part of the ongoing migration of API-side Babel plugins to Vite plugins.
